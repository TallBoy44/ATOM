# Cisco FMC PowerShell Module

*For Brandon Thomas*


## PowerShell Functions

**Login to Cisco FMC**

`Login-FMC`

**List Domains on FMC**

`Get-Domains`

**List ALL Devices within Domain**

`Get-Devices`

**List ALL SubInterfaces within Device**

`Get-SubInterfaces`

**Remove SubInterface from Device**

`Remove-SubInterfaces`
